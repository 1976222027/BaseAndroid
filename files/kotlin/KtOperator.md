## Kotlin常用操作符

| 属性 | 说明 | 备注 |
| ------------- |:-------------| :-----|
|?:|如果左侧非空返回左侧，否则返回右侧|当且左侧为空时，才会对右侧表达式求值|
|..|如果左侧非空返回左侧，否则返回右侧|当且左侧为空时，才会对右侧表达式求值|

## 2、..用法

~~~
for (i in 1..5) {   // 等价于 for (int i = 1 ; i <= 5 ; i++) { print(i) }
  print(i)  // 12345
}
这里的 1..5 左右都是闭区间的
~~~
## 3、until

~~~
for (i in 1 until 5) {
  print(i) // 1234
}
又有一些时候（大部分的时候）可能并不需要包括结束区间。那么，
这时候需要使用到 until 函数来替代 ..：
~~~
